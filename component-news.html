How to purchase ZEEW tokens?

5 May

ZEEW Token is a ERC20 Token, to participate in an Ethereum token sale you need a wallet that is smart contract compatible.

DO NOT PARTICIPATE TO TOKEN SALE DIRECTLY FROM ANY CRYPTOCURRENCY EXCHANGES USING THIS METHOD. YOU NEED A TOKEN COMPATIBLE ETHEREUM WALLET OTHERWISE YOUR FUNDS WILL BE LOST

We Accept ETH (Smart Contract, ETH from Exchange, BTC, LTC & Creditcard)

Participating in the token sale

  1. Buy ethers in a cryptocurrency exchange
  2. Transfer ether to a compatible wallet (see how to create myetherwallet)
  3. Create Zeew Wallet account here, then choose the payment method from inside the Zeew Wallet

Compatible wallets

The following hardware wallets are also known to be compatible with Ethereum tokens

  • MyEtherWallet (no download needed)
  • MetaMask (Firefox and Chrome browser addon) Recommended
  • Mist (Desktop)
  • Parity (Desktop)
  • Parity + Ledger (Hardware wallet)
  • imToken (iPhone)
  • imToken (Android)
  • Trust(iPhone)
  • Trust(Android)
  • Cipher(iPhone)
  • Cipher(Android)

The following hardware wallets are also known to be compatible with Ethereum tokens

  • MyEtherWallet + Ledger
  • MyEtherWallet + Trezor

MyEtherWallet: Example how to participate to ZEEW token sale

To participate an Ethereum token sale you need to fill your transaction with following details

  • Address — (Click here to get the address. ETH Smart Contract method only)
  • Data — (this is auto generated by MyEtherWallet in grey area, you do not need to enter it)

this is your personal reference number and it changes for every transaction. Do not share this with others. If you leave data field out your transaction is rejected. You need to click Advanced options to see this.

  • Gas limit: 250600 — gas tells that this is a smart contract transaction and is more expensive than normal Ether transfer. Note: Double check your the gas limit field value after filling Data field. MyEtherWallet may reset the gas value to an invalid number after inputting the data field.
  • Amount Fill in the Amount how many Ethers you wish to send, usually in a range 0.01–1000 ETH

AGAIN, DO NOT PARTICIPATE TO TOKEN SALE DIRECTLY FROM ANY CRYPTOCURRENCY EXCHANGES USING THIS METHOD, CHOOSE ANOTHER METHOD FOR ETH FROM EXCHANGE. YOU NEED A TOKEN COMPATIBLE ETHEREUM WALLET OTHERWISE YOUR FUNDS WILL BE LOST
